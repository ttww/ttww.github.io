Voltcraft-VC830

vc830 is a little C program which reads serial data from a voltcraft VC830 digital multimeter and transform it to different output formats

BNO055 AT STYLE DRIVER

This driver implements a simple "AT" style interface for the BNO055 sensor chip (BOSCH). The BNO055 is connected via the I2C interface

firmata4j

firmata4j is a client library of Firmata written in Java. The library allows controlling Arduino (or another board) which runs Firmata protocol from your java program (fork wirh esp wifi support)

ConfigurableFirmata

Firmata is a protocol for communicating with microcontrollers from software on a host computer

JavaFrameBuffer

ccess to Linux frame buffer devices from Java. E.g. for driving LCD SPI displays on the Raspberry Pi
